# React Learning Repository

This repository contains different React projects built while learning React fundamentals and modern development practices.

## Projects

### project-counter

A counter application demonstrating component state management and event handling in React. Built with Vite for fast development and ESLint for code quality.

### react-with-vite

A React project exploring basic component structure and setup with Vite as the build tool. Focuses on understanding the React development workflow.

### tailwind-props

A project exploring component composition and styling with Tailwind CSS. Contains a Card component that demonstrates handling props and CSS utility-based styling.

## Tech Stack

- React for UI components and state management
- Vite for fast build tooling and development server
- Tailwind CSS for utility-first styling
- ESLint for code linting and consistency

## Learning Focus

The projects in this repository focus on:

- Component lifecycle and state management
- Props and component composition
- Event handling and user interactions
- Modern tooling with Vite
- CSS techniques and styling approaches
- Code quality through linting

## Getting Started

Each project has its own package.json with specific dependencies. To run any project:

1. Navigate to the project directory
2. Install dependencies with npm install
3. Start the development server with npm run dev
